YouTube Enhancer Extension

This is a Chrome extension project. To use the extension:

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" in the top right
  3. Click "Load unpacked" and select the public/ folder

Project Structure

  • public/manifest.json - Extension configuration
  • public/background.js - Service worker for OAuth
  • public/content.js - Content script for YouTube pages
  • backend/ - Node.js API server

Setup Instructions

1. Backend: cd backend && npm install && npm start

2. Load extension from public/ folder in Chrome
